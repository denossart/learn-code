
==A PROGRAM

A set of instructions to send to a device


==A STATEMENT

A single instruction in a program


==A VARIABLE

A statement that tells a device to store data in memory

Unique name

Value might change over time


==FLOW CONTROL

Some statements change the flow of the program, i.e. the "if...else" statement
